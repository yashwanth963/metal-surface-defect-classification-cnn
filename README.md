# Metal Surface Defect Classification Using CNN

## Overview
A Convolutional Neural Network (CNN) developed for automated classification of steel surface defects.

## Technologies
- Python
- TensorFlow
- OpenCV
- NumPy
- Scikit-learn

## Dataset Classes
- Crazing
- Inclusion
- Patches
- Pitted Surface
- Rolled-in Scale
- Scratches

## Results
- Accuracy: 91%
- Evaluated using Precision, Recall, F1-Score, Confusion Matrix, and Precision-Recall Curves.
